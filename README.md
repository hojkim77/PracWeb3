## Getting Started

1. Ganache를 설치하고 실행합니다:

   ```bash
   # Ganache CLI 설치 (선택사항)
   npm install -g ganache

   # Ganache 실행
   ganache
   # 또는 Ganache GUI를 사용할 수도 있습니다
   ```

2. 개발 서버를 실행합니다:

   ```bash
   npm run dev
   # or
   yarn dev
   # or
   pnpm dev
   # or
   bun dev
   ```

3. [http://localhost:3000](http://localhost:3000)에서 앱을 확인합니다.

   로컬 환경에서는 Ganache가 자동으로 감지되어 Ganache 계정을 사용할 수 있습니다.
